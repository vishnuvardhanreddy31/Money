## ğŸ’¾ Data Management

### Local Storage
- All expenses are automatically saved to your browser's localStorage
- Data persists across sessions on the same device/browser

### Cross-Device Sync
Since data is stored locally, to use the app across multiple devices:
1. Click **Export** to download your expenses as a JSON file
2. On another device, click **Import** and select the JSON file
3. Your expenses will be merged (duplicates are avoided)

### Data Safety
- Regularly export your data as backup
- Keep exported JSON files in a safe location (cloud storage, email, etc.)
- Import your backup if you clear browser data or switch devices
